# ajieLib
一个轻量的原生JS库


## 安装与下载

### 使用npm

`npm install ajielib`

### 使用git

`git clone https://github.com/ajieOwO/ajieLib.git`


## 导入

```javascript
import 'ajielib';
```

## 模块介绍

- `ajieDate`：新增日期方法，封装了一个继承自Date的类AjieDate。
- `ajieArray`: 新增数组方法。
- `ajieXHR`：与服务器通讯。封装了get和post请求。
- `drawSector`： 饼图绘制。饼图内容严格按照传入数组中的顺序。
